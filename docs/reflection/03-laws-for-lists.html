Laws for Lists

We illustrate a case study for Liquid Haskell as a theorem prover, by proving the standard laws for lists. It is a formed versions of ยง 2.5 of "Towards Complete Specification and Verification with SMT".

Prelude

Overview is a Haskell module with Liquid Haskell annotations. We enable the Liquid Haskell flags that allow higher order reasoning and the reflection of all user-defined ADTs into the logic (both features are increasing the precision of verification and are off for efficiency). We also enable PLE globally, to automatically prove all the theorems. By default Liquid Haskell checks that all your functions are total.

module LawsForLists where {-@ LIQUID "--higherorder" @-} {-@ LIQUID "--exact-data-con" @-} {-@ LIQUID "--automatic-instances=liquidinstances" @-} import Language.Haskell.Liquid.ProofCombinators import Prelude hiding (length, head, tail, map, (++), (^))

First, we define a user-defined list with a termination metric length.

data L a = N | C {hd :: a, tl :: L a} {-@ data L [length] a = N | C {hd :: a, tl :: L a} @-} length :: L a -> Int {-@ length :: L a -> Nat @-} {-@ measure length @-} length N = 0 length (C _ xs) = 1 + length xs

Next, we define and reflect function composition and the standard list map and append functions.

{-@ infix ^ @-} {-@ reflect ^ @-} (^) :: (b -> c) -> (a -> b) -> a -> c (^) f g x = f (g x)
{-@ reflect map @-} map :: (a -> b) -> L a -> L b map _ N = N map f (C x xs) = f x `C` map f xs
{-@ infix ++ @-} {-@ reflect ++ @-} (++) :: L a-> L a -> L a N ++ ys = ys (C x xs) ++ ys = C x (xs ++ ys)

Finally, we use PLE to automatically prove the properties on lists.

  • Append Nil Identity
appendNilId :: L a -> Proof {-@ appendNilId :: xs:_ -> { xs ++ N = xs } @-} appendNilId N = () appendNilId (C _ xs) = appendNilId xs
  • Append Associative
appendAssoc :: L a -> L a -> L a -> Proof {-@ appendAssoc :: xs:_ -> ys:_ -> zs:_ -> { xs ++ (ys ++ zs) = (xs ++ ys) ++ zs } @-} appendAssoc N _ _ = () appendAssoc (C _ xs) ys zs = appendAssoc xs ys zs
  • Map Fusion
mapFusion :: (b -> c) -> (a -> b) -> L a -> Proof {-@ mapFusion :: f:_ -> g:_ -> xs:_ -> { map f (map g xs) = map (f ^ g) xs } @-} mapFusion f g N = () mapFusion f g (C _ xs) = mapFusion f g xs
  • Swapping
{-@ reflect swap @-} swap :: Ord a => L a -> L a swap (C x1 (C x2 xs)) | x1 > x2 = C x2 (C x1 xs) | otherwise = C x1 (C x2 xs) swap xs = xs
swapIdemp :: L Int -> Proof {-@ swapIdemp :: xs:L Int -> {swap (swap xs) = swap xs } @-} swapIdemp (C x1 (C x2 xs)) | x1 > x2 = () | otherwise = () swapIdemp xs = ()
